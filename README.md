# SpeechRecognition

'Go to Google' => Google.com<br>
'Go to Youtube' => Youtube.com<br>
'Go to Instagram' => Instagram.com<br>
'Go to Facebook' => Facebook.com<br>
'Go to GitHub' => Github.com<br>
